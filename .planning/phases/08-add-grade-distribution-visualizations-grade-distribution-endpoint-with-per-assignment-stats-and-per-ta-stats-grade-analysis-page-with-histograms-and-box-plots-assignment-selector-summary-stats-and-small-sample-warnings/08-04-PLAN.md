API contracts from Plan 02 endpoint (08-02):
```
GET /api/dashboard/grade-distribution/{course_id}
Response: { assignments: [{ assignment_id, assignment_name, points_possible, graded_count }] }
Sorted by graded_count DESC.

GET /api/dashboard/grade-distribution/{course_id}/{assignment_id}
Response: {
  assignment_id, assignment_name, points_possible,
  stats: { n, small_sample, mean, median, stdev, q1, q3, min, max },
  histogram: [{ bin_start, bin_end, count, label }],
  per_ta: [{ grader_name, n, mean }]
}
```
