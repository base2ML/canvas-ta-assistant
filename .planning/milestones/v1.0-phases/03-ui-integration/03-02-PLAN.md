The hook should:
1. Import `useCallback` and `useRef` from React, and `BACKEND_URL` from `../api`.
2. Export a named function `useSSEPost()` that returns `{ startPosting, cancel }`.
3. Maintain an `AbortController` ref via `useRef`.
4. `startPosting(assignmentId, requestBody, handlers)`:
   - `handlers` is an object with optional keys: `onStarted`, `onProgress`, `onPosted`, `onSkipped`, `onError`, `onDry_run`, `onComplete`, `onCancelled`.
   - Creates a new `AbortController`, stores in ref.
   - Calls `fetch` with method POST, Content-Type application/json, `JSON.stringify(requestBody)` body, and `signal` from controller.
   - If `!response.ok`, reads error JSON (fallback to statusText) and throws an Error.
   - Reads `response.body` via `getReader()` + `TextDecoder`.
   - Parses SSE by splitting on `\n\n`, extracting `event:` and `data:` lines.
   - For each parsed event, looks up handler by event name: capitalize first letter and prepend "on" (e.g., "started" -> "onStarted", "dry_run" -> "onDry_run"). Call handler with parsed JSON data.
   - Catches `AbortError` (err.name === 'AbortError') silently — this is normal cancel behavior.
5. `cancel()`: calls `abortRef.current?.abort()`.

Follow the exact SSE parsing pattern from RESEARCH.md Pattern 5 (lines 234-261 of research). Keep `buffer` for partial chunks. Use `{ stream: true }` in TextDecoder.decode.

```javascript
const handlePost = async () => {
  setShowConfirmDialog(false);
  setShowPreviewModal(false);
  setPosting(true);
  setPostResult(null);
  setPostError('');
  setPostProgress({ current: 0, total: 0, skipped: 0 });
  try {
    await startPosting(postAssignmentId, {
      course_id: String(currentCourse.id),
      template_type: 'penalty',
      user_ids: selectedStudentIds,
      override_comment: overrideComment || null,
      dry_run: isDryRun,
    }, {
      onStarted: ({ total }) => setPostProgress(prev => ({ ...prev, total })),
      onProgress: ({ current, total }) => setPostProgress(prev => ({ ...prev, current, total })),
      onPosted: () => {},
      onSkipped: () => setPostProgress(prev => ({ ...prev, skipped: prev.skipped + 1 })),
      onError: () => {},
      onDry_run: () => setPostProgress(prev => ({ ...prev, current: prev.current + 1 })),
      onComplete: (data) => setPostResult(data),
    });
  } catch (err) {
    if (err.name !== 'AbortError') {
      setPostError(err.message);
    }
  } finally {
    setPosting(false);
  }
};
```
