**Add populate_default_templates function** (new function, placed after init_db):

```python
def populate_default_templates() -> None:
    """Populate default comment templates if table is empty."""
    with get_db_connection() as conn:
        cursor = conn.cursor()
        cursor.execute("SELECT COUNT(*) as count FROM comment_templates")
        if cursor.fetchone()["count"] > 0:
            return  # Templates already exist

        default_templates = [
            {
                "template_type": "penalty",
                "template_text": (
                    "Late Day Update for this assignment:\n\n"
                    "Days late: {days_late}\n"
                    "Late days used (this assignment): {penalty_days}\n"
                    "Late days remaining: {days_remaining}\n"
                    "Penalty: {penalty_percent}%\n\n"
                    "Maximum late days per assignment: {max_late_days}\n\n"
                    "Please review the course late day policy if you have questions."
                ),
                "template_variables": json.dumps(
                    ["days_late", "penalty_days", "days_remaining", "penalty_percent", "max_late_days"]
                ),
            },
            {
                "template_type": "non_penalty",
                "template_text": (
                    "Late Day Update for this assignment:\n\n"
                    "Days late: {days_late}\n"
                    "Late days remaining: {days_remaining}\n\n"
                    "No penalty has been applied.\n\n"
                    "Maximum late days per assignment: {max_late_days}\n\n"
                    "Please review the course late day policy if you have questions."
                ),
                "template_variables": json.dumps(
                    ["days_late", "days_remaining", "max_late_days"]
                ),
            },
        ]

        for template in default_templates:
            cursor.execute(
                """INSERT INTO comment_templates
                   (template_type, template_text, template_variables)
                   VALUES (?, ?, ?)""",
                (template["template_type"], template["template_text"], template["template_variables"]),
            )
        conn.commit()
        logger.info(f"Populated {len(default_templates)} default comment templates")
```

**Template CRUD Functions:**

1. `create_template(template_type: str, template_text: str, template_variables: str | None = None) -> int`
   - INSERT into comment_templates, return new template id
   - Use `datetime.now(UTC)` for created_at and updated_at
   - Use `get_db_connection()` context manager (not transaction - single insert)

2. `get_templates(template_type: str | None = None) -> list[dict[str, Any]]`
   - SELECT all templates, optionally filtered by template_type
   - Order by template_type, then created_at DESC
   - Return list of dicts (using existing `dict(row)` pattern)

3. `get_template_by_id(template_id: int) -> dict[str, Any] | None`
   - SELECT single template by id
   - Return dict or None if not found

4. `update_template(template_id: int, template_type: str, template_text: str, template_variables: str | None = None) -> bool`
   - UPDATE comment_templates SET ... WHERE id = ?
   - Set updated_at to `datetime.now(UTC)`
   - Return True if row updated (cursor.rowcount > 0), False if not found

5. `delete_template(template_id: int) -> bool`
   - DELETE FROM comment_templates WHERE id = ?
   - Return True if deleted, False if not found

**History Recording Functions:**

6. `record_comment_posting(course_id: str, assignment_id: int, user_id: int, template_id: int | None, comment_text: str, status: str = "posted", canvas_comment_id: int | None = None, error_message: str | None = None) -> int`
   - INSERT with ON CONFLICT(course_id, assignment_id, user_id, template_id) DO UPDATE
   - On conflict, update: comment_text, canvas_comment_id, status, error_message, posted_at
   - Use `datetime.now(UTC)` for posted_at
   - Return the record id (cursor.lastrowid)
   - Log the posting attempt using `logger.info()` with course_id, assignment_id, user_id, status for audit trail (INFRA-09)

7. `get_posting_history(course_id: str, assignment_id: int | None = None, status: str | None = None, limit: int = 100) -> list[dict[str, Any]]`
   - SELECT with optional filters for assignment_id and status
   - Build query dynamically with parameterized WHERE clauses (follow get_sync_history pattern)
   - Order by posted_at DESC, LIMIT by limit param
   - Return list of dicts

8. `check_duplicate_posting(course_id: str, assignment_id: int, user_id: int, template_id: int | None) -> dict[str, Any] | None`
   - SELECT from comment_posting_history WHERE course_id = ? AND assignment_id = ? AND user_id = ? AND template_id = ? AND status = 'posted'
   - Return the existing record dict if found, None if no duplicate
   - This enables Phase 2 to check before posting without relying solely on the UNIQUE constraint

All functions should use `get_db_connection()` context manager. Use parameterized queries exclusively (no string concatenation). Follow the existing code style: type hints, docstrings, `dict(row)` for results.
