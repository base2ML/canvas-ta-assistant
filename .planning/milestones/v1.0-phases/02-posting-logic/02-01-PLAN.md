**3. calculate_late_days_for_user function** (add after render_template):

```python
def calculate_late_days_for_user(
    user_id: int,
    assignment: dict[str, Any],
    submissions: list[dict[str, Any]],
    max_late_days: int = 7,
) -> dict[str, Any]:
```

This function calculates late day variables for a single student on a single assignment. Reuse the same logic as `get_late_days_data` endpoint (grace period, ceiling division, etc.):

- Find submission for this user_id and assignment in submissions list
- Parse submitted_at and due_at using dateutil_parser
- Apply LATE_SUBMISSION_GRACE_PERIOD_MINUTES (15 min) grace period
- Calculate days_late using math.ceil(total_seconds / 86400)
- Calculate penalty_days = min(days_late, max_late_days)
- Calculate penalty_percent = penalty_days * 10 (10% per day, matching existing default template logic)
- Calculate days_remaining = max(0, max_late_days - penalty_days) -- this is per-assignment remaining, NOT cumulative across assignments (matches template variable semantics)
- Return dict with keys: days_late, days_remaining, penalty_days, penalty_percent, max_late_days
- If no submission or not late, return: days_late=0, days_remaining=max_late_days, penalty_days=0, penalty_percent=0, max_late_days=max_late_days

**5. Preview endpoint**:

```python
@app.post("/api/comments/preview/{assignment_id}")
async def preview_comments(assignment_id: int, request: PostCommentsRequest) -> PreviewResponse:
```

- Call validate_posting_safety(request.course_id) -- raise 403 if unsafe
- Get assignment from db.get_assignments(request.course_id), find by assignment_id; raise 404 if not found
- Get submissions via db.get_submissions(request.course_id, assignment_id)
- Get users via db.get_users(request.course_id)
- Resolve template via resolve_template() (skip if override_comment provided)
- Read max_late_days from settings via db.get_setting("max_late_days_per_assignment"), default 7
- For each user_id in request.user_ids:
  - Look up user name from users list (fallback: f"User {user_id}")
  - Call calculate_late_days_for_user() to get variable data
  - If override_comment: use that text directly; else: call render_template() with template text and variable data
  - Check duplicate via db.check_duplicate_posting(course_id, assignment_id, user_id, template_id)
  - Determine template_type from resolved template or None if override
  - Build CommentPreview object
- Return PreviewResponse with assignment_name from assignment data
